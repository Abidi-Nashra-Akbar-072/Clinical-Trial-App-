# Clinical Trial Matching System

A Streamlit application for matching patients with relevant clinical trials using AI embeddings and FAISS similarity search.

## Features
- ClinicalTrials.gov API integration
- Semantic search using Bio_ClinicalBERT
- FAISS vector similarity
- Interactive Streamlit UI
- CSV export

## Installation

```bash
pip install -r requirements.txt
```

## Run App

```bash
streamlit run app.py
```
